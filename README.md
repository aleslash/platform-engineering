# platform-engineering

This is a repo that contain topics and guides about platform engineering and platform engineering study paths.

## Contents

This repo contains the following directories:

- `golang/`: Contains resources and guides related to the Go programming language.
- `certification/`: Contains information and resources for various certifications from platformengineering.org
- `architecture/`: Contains resources and guides on platform architecture.
- `auth/`: Contains resources and guides on authentication mechanisms and best practices.
- `argo-stack/`: Contains resources and guides related to the Argo Stack.
- `service-mesh/`: Contains resources and guides on service mesh technologies.
- `gateway-api/`: Contains resources and guides on Gateway API.
- `networking/`: Contains resources and guides on networking concepts and practices.
- `gitops/`: Contains resources and guides on GitOps practices and tools.
- `cloud/`: Contains resources and guides on cloud computing and related technologies.
- `crossplane-ack/`: Contains resources and guides related to Crossplane and ACK (AWS Controllers for Kubernetes).
